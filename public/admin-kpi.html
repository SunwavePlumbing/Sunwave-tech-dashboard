Server API endpoints (advanced)

These are the JSON endpoints the dashboard fetches from. Listed for reference — you usually don’t need to hit them directly.

  • GET /api/metrics?range=mtd — technician leaderboard
  • GET /api/marketing — marketing 12-month history
  • GET /api/owners-financial — P&L data
  • GET /api/qbo-balance — balance sheet + bank history
  • GET /api/qbo-marketing — QBO marketing spend
  • GET /api/qbo-reconciliation — QBO recon status
  • GET /api/qbo-accounts — available P&L accounts
  • GET /api/account-detail?acct=...&month=... — expense drill-down
  • GET /api/diagnostics/coverage?range=mtd — coverage reconciliation (password-gated)
  • GET /api/diagnostics/kpi — KPI investigation data (password-gated)
  • GET /api/debug/techs — post-migration data audit
  • POST /api/kpi/report-issue — submit a tech issue report
  • GET /api/kpi/recent-reports — recent issue reports (public)
  • GET /api/kpi/admin/issues — all issue reports (admin)
  • POST /api/kpi/admin/issues/:id/status — resolve/dismiss/re-open
  • GET /api/kpi/admin/job/:id — HCP job + active reconciliation
  • POST /api/kpi/admin/reconcile — save a reconciliation
  • DELETE /api/kpi/admin/reconcile/:jobId — remove a reconciliation
  • GET /api/kpi/admin/reconciliations?month=YYYY-MM — list reconciliations
